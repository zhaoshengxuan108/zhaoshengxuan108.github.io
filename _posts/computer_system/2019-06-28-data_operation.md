# 数据及运算

## 数据及概念

1. **bit**: bit是最小运算单元，在计算机中占一个存储单元。
2. **字节**：8个bit组成一个字节
3. **位值记数法（positional notation）**: 一种表达数字的系统，数位按顺序排列，每个数位有一个位值，数字的值是每个数位和位值的乘积之和。
4. **无符号整数**
5. **有符号整数**
6. **反码**：将一个正数所有bit全部取反。例：原码：00100110  反码：11011001
7. **补码**：将一个正数所有bit全部取反，再加1。 例：原码：00100110  反码：11011001 补码：11011010
